This file describes the hardware connections between rapsberry, gpio expander and printer parallel port.
It should reflect the software configuration at each commit.

## PCF8574A: ##

A0 -- A2 --> GND
SDA --> raspy.SDA
SCL --> raspy.SCL
P0 -- P7 --> printer.D0 -- printer.D7

## Printer: ##

DB25 pin 18 (GND) --> GND
DB25 pin 17 (SelectIn) --> GND
DB25 pin 1 (Strobe) --> raspy.GPIO_4
DB25 pin 11 (Busy) --> raspy.GPIO_17


max output valid from clock 3.45 uS (set delay to 100uS)
